Nothing but my very own express boilerplate

## :sparkles: Features

-   Tests with Jest
-   Continuous integration with CircleCI
-   Code coverage with Codacy
-   JS Linting with eslint, prettier and Airbnb style guide

## Install dependencies

```bash
yarn
npm install
```

## Start server

```bash
yarn dev
npm run dev
```

## Linting

```bash
yarn lint
npm run lint
```

### Error fixing

```bash
yarn lint:fix
npm run lint:fix
```

## Testing

```bash
yarn test
npm run test
```
